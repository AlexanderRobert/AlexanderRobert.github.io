Projects

Detailed description of the course projects

  • Project 1

The first project consists of:

1. Build an individual website with html in either English or Icelandic and document your work

2. Upload your website to github using Git Bash

3. Post it as a comment, besides your name, under the class google docs sheet

  • Project 2

The second project consists of:

1. Use a CAD software to build a parametric design of a construction kit

2. Cut your design with a laser cutter

3. Document the process and upload your design files

  • Project 3

The third project consists of:

1. Use a design software to make something that could not be made subtractively, and 3D print your design

2. 3D scan something with 3D ph

3. Document the process and upload your design files

  • The final project

The final project consists of:

1. Plan a final project that integrates some of the course subjects

2. Your project should integrate some 2D and 3D elements

3. Determine an intellectual property licensing

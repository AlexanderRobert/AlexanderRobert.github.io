The first project consists of:

The approach

The team decided to use project 4: Make something big and project 5: Molding and casting to supplement this project. In total, those two projects combined should cover the design and manufacture phases of the final project and that way the team is integrating course work into the project.

Molding and casting

The first thing we did was to go online and find ideas. The group liked the following idea:

Now that we knew the design we needed to know certain dimensions about a classic wine glass. We found the following image on google.
The bottom plate is where the wine glass holders will be fastened. The idea is to create a mold in a similar shape as these ones. The current idea is either to drill the molded wine glass holders to the bottom plate or glue it.
First up we draw the bottom plate.
Then I created the form of the wine glass holders.
Now we want to create the mold. Note that it's only needed to create a mold for a single support and then we simply create four supports from that mold. We started by creating a box bigger than the support.
Then we used the command combine and selected the box as the target body and the support as the tool body. The operation was set to cut. This command did used the target body as a cutting profile of the target body. Then we checked the box "Keep Tools". That makes sure that our tool body doesn't disappear.
Then we split the box into two halves so to create a two piece mold.
Then we created a hole into the top half of the box so that material could be poured into the mold.
In the next course session we asked the tutor for comments on the mold and he pointed out that since my support has a flat surface at the bottom, it would be a much better format to put the mold upside down and keep it open. So the mold doesn't have to be a two piece. So we decided to undo a lot of changes in the timeline and went back to the part where we only had the one support selected. We began by turning the support upside down.
So instead of creating a box around the support, we decided to create a box only around the bottom and the sides. We created a sketch on the support surface and constrained it so that the support was in the middle and that the highest part of the box was collinear with the support.
Then we extruded the sketch so that it would be as long as the support.
We created a sketch on the surface of the support so we could extrude the materials 10mm more on each side. We made sure that the operation was set as "New Body" instead of "Join" because we wanted this extra material to become a part of the mold but not the support.
We used the command "Combine" to make the new material become a part of the mold.
Now when we moved the support out of the mold we could see that the mold was ready!

The only thing left to do is to cut out this mold and pour the molding material into the mold. Gravity will do its work to make sure that the top surface becomes flat.

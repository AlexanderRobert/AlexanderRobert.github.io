Project 1

Documentation

The project description

The first project consists of:

1. Build an individual website with html in either English or Icelandic and document your work
2. Upload your website to github using Git Bash
3. Post it as a comment, besides your name, under the class google docs sheet

Assignment completion checklist:

1. Have you made a website and described, step by step, how you did it
2. Have you introduced yourself and made a short but relevant resume
3. Have you documented the steps needed to upload the website
4. Have you introduced your group
5. Have you organized how you will manage your final project
6. Have you made a proposal for you final project
7. Have you made your code (in text format) accessible on your website
8. Have you provided relevant links you used and included all search "engine key words" you used

The project

The website used Brackets for programming and GitHub for version control. HTML5 UP was also used for templates on the website. A lot of time was spent filling out the website and deciding what to put in it and how it should be set up. Google was widely used to find solutions to various programming problems.

An introduction about myself can be found in the about me section of the website.

The steps needed to upload the website are as follows:
1. Download Brackets
2. Download Github
3. Download a website template from HTML5 UP
4. Fill in the website with all the relevent information
5. Finish your design and make simple changes if needed

An introduction about the group can be found in the about the team section of the website.

The final project organization as well as the description of a proposal of the final project can be seen in the final project section in the project section of the website.

The text format of the code can be found in the code section of the website.
